API Microservices Bundle

A collection of API microservices, written to complete the Back End Development and APIs certification provided by FreeCodeCamp

TimeStamp

Project Details

Example Usage:

  • /api/timestamp
  • /api/timestamp/2015-12-25
  • /api/timestamp/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

URL Shortener

Project Details

Example Output:

{"original_url":"https://www.freecodecamp.org",
"short_url":"123"}

Example Usage:

/api/shorturl/123

Will redirect to:

https://www.freecodecamp.org

Exercise Tracker

Project Details

Example Usage:

Use the form below to add a new user to the database. Use the second form to add exercises to the user by id.

/api/users to access a list of all users

/api/users/:id/logs to access a particular user's logs

Accepts from, to, and limit queries.

E.g., /api/users/:id/logs?from=2022-06-12&to=2022-07-15&limit=3

Create New User

Add Exercises

*Optional (will default to today's date)
